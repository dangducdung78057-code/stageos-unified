## 1. 色彩规则引擎

三要素的舞台分工：**色相=情绪基调，明度=氛围轻重，饱和度=情绪强度**。

### 1.1 节目类型 → 色彩参数

| 节目类型 | 色相 | 明度 | 饱和度 | 情绪目标 |
| --- | --- | --- | --- | --- |
| 童声合唱·欢快 | 橙/黄 | 高 | 中高 | 温暖、天真 |
| 童声合唱·抒情 | 蓝/紫 | 中低 | 低 | 宁静、深情 |
| 民族舞蹈 | 红/绿/金 | 中高 | 高 | 热烈、生命力 |
| 现代舞 | 灰/白/黑 | 中 | 低 | 纯粹、哲思 |
| 戏剧/朗诵 | 暖棕/深红 | 中低 | 中 | 庄重、沉浸 |
| 军旅题材 | 军绿/卡其 | 中 | 中低 | 阳刚、纪律 |

### 1.2 配色策略

- **同类色**（色相环 15° 内）：合唱、朗诵等统一感节目；推荐同色相不同明度叠穿（深蓝+浅蓝+白）。
- **互补色**：需要视觉冲击时——蓝↔橙（力量/史诗）、黄↔紫（梦幻）、红↔绿（节日，须降饱和防刺眼）。
- **符号色**：低饱和整体中一处高饱和点缀（全白中一抹红），作为视觉记忆点。

### 1.3 色温-空间法则

暖色收拢空间（教室/小舞台主光用暖）、冷色拓展空间（大厅背景光用冷）。色温四档：1000–2000K 烛光怀旧 / 3000–4000K 日常舒适 / 5000–6000K 正午现实 / 7000K+ 冷峻科幻。

## 2. 色彩情感词典（情绪 → 色）

| 色 | 核心情感 | 标签 |
| --- | --- | --- |
| 红 | 激情、危险、权力 | #power |
| 粉 | 纯真、甜蜜 | #innocence |
| 橙 | 温暖、社交、青春 | #warmth |
| 黄 | 天真、不安、警示 | #naive |
| 绿 | 自然、或黑暗/腐败 | #nature |
| 紫 | 幻想、神秘 | #fantasy |
| 蓝 | 冷静、忧郁、理智 | #calm |

组合情感：粉+橙+紫=浪漫；明亮高饱和=青春活力；冷灰=悬疑压抑；金属蓝=科幻；紫蓝=梦幻超现实；红+金=喜庆中国；黑白=经典戏剧。
用法：分镜表每段定情绪后，先查本表定向，再去 color-script.md §2 选 hex 组。

## 3. 布光法则与模板

### 3.1 四变量速查

| 变量 | 选项 | 舞台效果 |
| --- | --- | --- |
| 强度 | 强光=现实清晰；弱光=神秘沉浸；逆光=剪影仪式 | 抒情用弱，写实用强 |
| 方向 | 顺光=全员清晰（合唱）；侧光=立体（独唱/独舞）；逆光=高潮剪影；顶光=威压（反派/紧张） | 按角色地位配方向 |
| 软硬 | 硬光=冲突/军旅/力量；柔光=抒情/儿童/温馨 | 雾/纱/柔光箱可化硬为柔 |
| 冷暖 | 暖=前景/聚焦/亲密；冷=背景/疏离/空旷 | 大小舞台公式见 §1.3 |

### 3.2 两个经典布光模板

- **卡拉瓦乔式（酒窖光）**：主体强光、背景暗而有细节 → 独唱/独舞/仪式高潮，需聚光灯+暗场。
- **伦勃朗式（三角光）**：侧上 45° 主光、面部三角光斑 → 领唱/朗诵的面部塑造，hall-800 面光区的默认打法。

### 3.3 天气光（室外演出模拟）

正午=硬光高对比（需柔化）；黄昏=暖色低角度（**最佳演出/拍摄时段**）；阴天=柔光压抑；雾=神秘；雪=冷白神圣。室外时段选择优先查 scene-light 实算，勿凭感觉。

## 4. 服化道规则

### 4.1 服装色彩叙事（《小丑》法则）

情绪/角色转折 = 服装色彩切换：前段低饱和灰暗（困境）→ 后段高饱和鲜明（爆发）。例：合唱《我和我的祖国》抒情段白/浅蓝 → 激昂段红/金。对应 color-script.md §1 叙事弧的"高潮跳变"规则。

### 4.2 材质速查

丝绸缎面=光泽流动（民族/古典，防反光过强）；棉麻=自然朴素（田园/现代）；纱/雪纺=飘逸（梦幻，注意透光）；亮片金属=夺目（开场，控制面积）；皮革 PVC=硬朗未来（科幻，闷热噪音）。**铁律：面料必须过麦克风静音测试。**

### 4.3 化妆与道具

- 舞台妆比日常妆浓 30–50%（灯光吃妆）；暖腮红唇色=健康亲近（儿童），冷眼影=神秘距离（现代/反派）。
- 道具五规：必有叙事功能；颜色入整体配色；无尖锐易碎；大道具算搬运；发光/反光件必须在舞台灯下实测。

## 5. 构图与队形

| 电影构图 | 心理 | 队形应用 |
| --- | --- | --- |
| 对称 | 安心、庄严、童话感 | 合唱弧形/八字，指挥为轴，左右严格对称 |
| 中心 | 聚焦、神圣 | 领唱/独舞居中环绕 |
| 倾斜 | 紧张、冲突 | 舞蹈冲突段不规则站位 |
| 边缘 | 疏离、孤独 | 戏剧角色边缘化 |
| 框架 | 窥视、保护 | 借舞台拱门/侧幕做画框 |

合唱队形声学×视觉：弧形（经典/比赛）、台阶分层（大团）、V 字（有领唱）、圆（小团互动）。舞蹈：方阵（力量）、斜线（动感）、圆/半圆（民族/抒情）、S/波浪（古典）、高低层次（群舞高潮）、分散聚合（情节叙事）。
2.5D 工具方向（已列入 roadmap）：对称检测、黄金分割辅助线、画面定格预览。

## 6. 节目类型视觉规范库

```yaml
program_types:
  童声合唱_欢快:  {color_scheme: [橙, 黄, 粉], saturation: 中高, lighting: 暖光柔光, costume: 明亮轻盈, formation: [弧形, V字]}
  童声合唱_抒情:  {color_scheme: [蓝, 紫, 白], saturation: 低, lighting: 冷光柔光+逆光, costume: 白/浅色长裙, formation: [弧形, 台阶]}
  民族舞蹈:       {color_scheme: [红, 绿, 金], saturation: 高, lighting: 暖硬光+色彩光, costume: 民族重工, formation: [圆, 方阵, 高低]}
  现代舞:         {color_scheme: [灰, 白, 黑], saturation: 低, lighting: 冷光+聚光/追光, costume: 简约贴身, formation: [不规则, S形]}
  戏剧_校园剧:    {color_scheme: [随剧情], saturation: 中, lighting: 场景化, costume: 角色化, formation: [场景化]}
  朗诵:           {color_scheme: [藏青, 墨绿, 深色系], saturation: 低, lighting: 暖面光+伦勃朗光, costume: 正装/改良中式, formation: [一字, 半圆]}
  军旅题材:       {color_scheme: [军绿, 卡其], saturation: 中低, lighting: 硬光高对比, costume: 军装/迷彩, formation: [方阵, 直线]}
```

未列品类（器乐/戏曲等）按"类型片迁移"原则类推：力量型→动作片（快调度+橙蓝对比）、科技主题→科幻片（冷蓝+暖橙+发光道具）、悬疑→惊悚片（暗场+追光+错位）、抒情→爱情片（柔暖+纱花）。

## 7. 与现有技能的对接裁定

1. **色温两套刻度不冲突**：本文件 §1.3 是人工光源刻度；sun_position.py 输出的 3200/4300/5600K 是自然光分段。室外以实算为准，本表仅作人工补光参考。
2. **与 screen_tint.py 对应**：§3.2 伦勃朗面光 → front_fill 区；轮廓剪影 → rim 区；空间基调 → ambient；地面氛围 → floor_bounce。屏色仍按 color-script.md §3 执行。
3. **品类映射**：本库 7 类规范 ↔ storyboard_lint.py 的 8 品类规格按名称映射，未命中者走 §6 类推规则；lint 只管人数/时长红线，不管美学。
4. **优先级**（报告结论，采纳）：色彩 > 灯光 > 服装 > 构图——预算受限时按此顺序砍。
